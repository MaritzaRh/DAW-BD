¿Qué es ODBC y para qué es útil?

Open DataBase Connectivity (ODBC) estandar de acesso a bases de datos, es util para crear una capa intermedia entre la aplicacion y el DBMS de manera que sin importar que DBMS se emplee se pueda acceder a la informacion.

¿Qué es SQL Injection?

Tecnica de insercion de codigo que busca destruir una BD o robar informacion de ella por medio del request de input a un usaurio donde este responde con codigo SQL que es interpretado y respondido por la aplicacion

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Validando y sanitizando los inputs que se le solicitan al usuario, usando Query predeterminados o estandarizados, convirtiendo los caracteres especiales, proteger instrucciones de busqueda, verificar tamaño y tipo de datos, no usar sql construido dinamicamente.

¿Por qué es una buena práctica separar el modelo del controlador?

Ya que de esta manera se asegura un poco que la informacion no este a simple vista, es decir que cualquier persona pueda entrar directo a modificarla e intentar alterar la aplicacion de manera ilicita, a la vez esta establecido que la ide ade HTML 5 es mantener cada capa totalmente separa con el fin de brindar una menjor mantenibilidad al codigo, hacerlo mas estrucgturado y finalmente mas seguro

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

Actualmente AJAX es considerada la principal y mas sencilla tecnologia para la creacion de RIA's debido a que el concepto clave de una RIA es el no recargar la pagina y solo obtener informacion de la BD por medio de requests por ellO AJAX es basico , ya que es el conjunto de tecnologias que resuleven este problema y les da la caracteristica basica a estos.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

CORS, SQLi, XSS; De ambos lados tanto para sanitizar entradas como para validar Sitios que hacen request al nuestro, siempre sera mejor tener validado ambos lados
