¿Por qué es una buena práctica separar el controlador de la vista?.

Porque en caso de mantenimiento, es más sencillo acceder sólo a la parte que se necesita modificar, en lugar de tener que buscar dentro de un enorme bloque de código.

Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?
  • $_ENV Es una variable tipo array asociativo de variables pasadas al script actual a través del método del entorno.
  • $_SERVER Es un array que contiene información como cabeceras, rutas y ubicaciones de script. Tales entradas son creadas por el servidor web.
  • $_FILES Es un array asociativo de elementos subidos al script en curso a través del método POST
  • $_COOKIE Es una variable tipo array asociativo de variables pasadas al script actual a través de Cookies HTTP.
  • $_SESSION Es un array asociativo que contiene variables de sesión disponibles para el script actual.
  • $_REQUEST Es un array asociativo que por defecto contiene el contenido de $_GET, $_POST y $_COOKIE.

¿Qué es XSS y cómo se puede prevenir?

XSS (Cross-Site Scripting) es un tipo de "ataque" que puede sufrir una aplicación web cuando una tercera persona "inyecta" código javascript o de otro lenguaje en la página web, evitando las medidas de control. Para prevenir esto, se pueden utilizar funciones de PHP como htmlspecialchars(), htmlentities() y strip_tags().
