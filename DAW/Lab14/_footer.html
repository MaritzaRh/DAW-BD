¿Qué es ODBC y para qué es útil?

Open DataBase Connectivity (ODBC) estandar de acesso a bases de datos, es util para crear una capa intermedia entre la aplicacion y el DBMS de manera que sin importar que DBMS se emplee se pueda acceder a la informacion.

¿Qué es SQL Injection?

Tecnica de insercion de codigo que busca destruir una BD o robar informacion de ella por medio del request de input a un usaurio donde este responde con codigo SQL que es interpretado y respondido por la aplicacion

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Validando y sanitizando los inputs que se le solicitan al usuario, usando Query predeterminados o estandarizados, convirtiendo los caracteres especiales, proteger instrucciones de busqueda, verificar tamaño y tipo de datos, no usar sql construido dinamicamente.
