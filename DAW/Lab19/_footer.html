¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

Actualmente AJAX es considerada la principal y mas sencilla tecnologia para la creacion de RIA's debido a que el concepto clave de una RIA es el no recargar la pagina y solo obtener informacion de la BD por medio de requests por ellO AJAX es basico , ya que es el conjunto de tecnologias que resuleven este problema y les da la caracteristica basica a estos.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

CORS, SQLi, XSS; De ambos lados tanto para sanitizar entradas como para validar Sitios que hacen request al nuestro, siempre sera mejor tener validado ambos lados



¿Qué es JSON?

JavaScript Object Notation, lenguage de intercambio de datos, es desarrolla en un formato ligero de texto el cual es legible para humanos y computadoras, a la vez este tiene similitudes con Python, Java o C en cuanto a sintaxis y finalmente se puede encontrar como objeto, arreglo, valor, numeros
