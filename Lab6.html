Preguntas a responder


  • ¿Cuáles son las diferencias entre los posibles valores de la propiedad position?

    • static: Es el valor por default. Los elementos se posicionan conforme al flujo normal del documento.

    relative: Los elementos se posicionan de manera relativa a su posición normal.

    fixed: Los elementos se posicionan de manera relativa a la ventana del navegador, es decir, siempre se mantienen en el mismo lugar.

    absolute: Los elementos se posicionan de manera relativa a la posición del elemento antecesor más cercano.

  • ¿Cuáles son los valores estándar para la propiedad visibility?

    • visible: Es el valor por default. El elemento está visible.

    hidden: El elemento es invisible, pero sigue ocupando su espacio.

    collapse: Sólo para elementos de una tabla. Remueve una fila o columna, sin afectar el diseño de la tabla. El espacio que queda después de remover la fila o columna, queda disponible para otro contenido.

  • ¿Qué es el zIndex y para qué sirve?

    • Es una propiedad que especifica el orden de un elemento dentro de un stack. Los elementos con un zIndex mayor, se colocan siempre en frente de aquellos con un zIndex menor.
