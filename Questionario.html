Laboratorio 4.1, 22 enero, 2018




Questionario


  • ¿Qué es entonces una base de datos - DB?

    • Una base de datos es una colección de archivos interrelacionadosque nos permite almacenar información de manera organizada y controlada.

  • ¿En qué casos es conveniente usar bases de datos?

    • Para todo aquél organismo que requiera de optimizar y guardar requerimientos de información como lo es en las ventas, compras, finanzas, etc; para que resulten más eficientes los procesos, sobretodos si se trata de información de gran volumen[1].

  • ¿Qué es un sistema de gestión de base de datos - DBMS?

    • Un Sistema de Gestión de Base de Datos (DBMS por sus siglas en inglés) es una colección de archivos interrelacionados y un conjunto de programas para acceder a estos últimos. La gestión de datos implica tanto la definición de estructuras para el almacenamiento de información como la provisión de mecanismos para la gestión de la información.

  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.

    • 1. Interacción con el gestor de archivos.

    2. Implantación de la integridad. Se debe satisfacer la integridad.

    3. Implantación de la seguridad. Esto incluye la recuperación de información y la privacidad de la misma.

    4. Copia de seguridad y recuperación. Un buen sistema debe responder a los requerimientos implícitos, esto quiere decir que la informción debe contar con una copia de seguridad constante y recuperable en caso de cualquier fallo humano, del sistema mismo o por causas naturales como una posible tormenta.

    5. Control de concurrencia. El gestor de base de datos tiene como responsabilidad establecer la interacción entre los usuarios concurrentes[2].

  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

    • Cuando estuve laborando en una Universidad, hubiese sido conveniente utilizar múltiples bases de datos así como un gestor para manejar toda la información no sólo porque es compleja y grande, sino que, sobretodo, hacían falta copias de seguridad debido a que la información guardad en Excel solía perderse con facilidad, por lo que el manejo de tiempo no era eficiente ni bien aprovechado.
