  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • El objetivo principal de java es la creación de aplicaciones mediante el lenguaje orientado a objetos, mientras que Javascript es la interactividad de páginas web basado en prototipos. Por otro lado, Java es compilado y Javascript un lenguaje interpretado. No obstante, su sintaxis es similar, y aunque no deriva el uno del otro, puede ser más sencillo entender uno si se sabe el otro [1].
